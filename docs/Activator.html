Class: Activator

Activator(container)

new Activator(container)

Turn an element into an clickToUse element. When not active, the element has a transparent overlay. When the overlay is clicked, the mode is changed to active. When active, the element is displayed with a blue border around it, and the interactive contents of the element can be used. When clicked outside the element, the elements mode is changed to inactive.
